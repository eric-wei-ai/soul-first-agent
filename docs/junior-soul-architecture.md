# Junior Soul Architecture — Four-Layer Design for AI Employees

## Four Layers

1. **Soul Layer** (SOUL.md) — Values, identity, boundaries
2. **Skill Layer** (tools, skills) — What the agent can do
3. **Memory Layer** (memory/) — What the agent knows
4. **Security Layer** (SECOPS.md) — What the agent must check before acting

## Design Principle
Build from layer 1 down. Never deploy an agent with skills but no soul.
