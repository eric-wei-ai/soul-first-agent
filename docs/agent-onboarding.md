# Agent Onboarding — Safely Introducing External Agents

## Principle
Treat new agents like new employees: verify credentials, limit access, monitor behavior.

## Checklist
1. Audit the agent's source code before installation
2. Check download count and community feedback
3. Never install agents from untrusted sources
4. Start with minimal permissions, expand as trust builds
5. Monitor for unexpected network calls or file access
